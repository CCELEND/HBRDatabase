## 安装

### 最新的 python3

### opencv-python、Pillow

```sh
pip install opencv-python Pillow
```

镜像加速（建议使用）

```sh
pip install -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple opencv-python pillow
```

运行 HBRDatabaseGUI.py 即可，更新失败的话多更新几次
